# Linux常用命令

从指定url下载资源

yum install -y wget
wget http://download.redis.io/releases/redis-5.0.5.tar.gz

解压tar.gz

tar xf xxx.tar.gz

Linux开机启动项目录

/etc/init.d

Linux环境变量

# 修改环境变量
vi /etc/profile
# 重启环境变量
source /etc/profile

根据进程名查看运行进程

ps -ef | grep redis

nc命令

# 安装包
yum install -y nmap-ncat
# 监听TCP/UDP端口
nc localhost 6379

全局搜索文件

find / -name "*.err"

netstat

netstat -anp|grep 5000
# 查看Tcp连接
netstat -natp

根据PID终止进程

kill -9 {pid}

查看机器负载

df -h

SpringBoot 后台启动

nohup java -jar myproject.jar >myproject.log 2>&1 &

命令详解：
nohup：不挂断地运行命令，退出帐户之后继续运行相应的进程。
>myproject.log：控制台输出到myproject.log日志文件中。
2>&1：标准错误重定向到标准输出。
最后的&：让改作业在后台运行。

查看进程数

pstree

管道

  1. 衔接：前一个命令的输出作为后一个命令的输入
  2. 管道会触发创建 子进程
# $$ 优先级高于 |
echo $$ | more
echo $BASHPID | more

父子进程

/bin/bash 开启一个子进程

父子进程间数据是隔离的，但是父进程可以使用 export 使子进程看到数据

export 的环境变量，子进程修改不会影响父进程，父进程修改也不会影响子进程
